Snapshot Update Site

This is the Eclipse update site for the Eclipse extensions developed in the Cooperate project.

Attention: This site provides the latest snapshot builds of the extensions, which might be incomplete or unstable.

Usage

In order to use the extensions, please use the regular Eclipse update site mechanism as follows:

  1. Open the installation dialog via Help - Install new Software
  2. Add this page URL to the update site list
  3. Tick the Contact all update sites during install to find required software option
  4. Select the features you want to install
